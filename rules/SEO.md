SEO ВНУТРИ <html> … </html> (ON-PAGE)
2.1. <title>

 Есть на каждой странице

 Уникальный

 ≤ 60–65 символов

 Не дублируется между страницами

 Не совпадает 1 в 1 с H1

2.2. <meta description>

 Есть на каждой странице

 Уникальный

 ≤ 150–160 символов

 Не автоспам

 Не пустой

2.3. <meta robots>

 index, follow — для SEO-страниц

 noindex — для служебных / мусорных

 Нет конфликтов (страница в sitemap + noindex ❌)

2.4. H1

 Ровно один H1 на страницу

 H1 отражает суть страницы

 H1 не размножается в сквозных блоках

 Нет пустого H1

2.5. H2–H6

 Логичная иерархия

 Нет пропусков уровней (H2 → H4 ❌)

 Не используются для дизайна

 Нет одинаковых SEO-заголовков на всех страницах

2.6. HTML-структура

 Один <main>

 <header>, <footer> используются корректно

 Нет скрытого контента (display:none для SEO-текста)

 Нет контента, загружаемого только JS

2.7. Внутренние ссылки

 Все ссылки кликабельные (<a href>)

 Нет ссылок с href="#" для SEO-блоков

 Нет битых внутренних ссылок

 Анкоры читаемые, не “Подробнее”

2.8. Изображения

 alt у всех смысловых изображений

 Нет SEO-спама в alt

 Изображения не подгружаются только через JS

2.9. Сквозные блоки

 Нет одинаковых SEO-H1/H2 на всех страницах

 Нет повторяющихся SEO-текстов

 Сквозные блоки — нейтральные

2.10. Локальное SEO (on-page)

 Адрес текстом

 Телефон текстом

 Название клиники единообразно

 Контакты доступны с любой страницы
