Читай ruLes/ и следуй правилам вёрстки:

— Не дублируй стили и классы. Если класс уже существует, дополняй его, а не создавай повторно.
— Если у родительского элемента уже заданы стили (цвет, шрифт, размеры и т.п.), не повторяй эти свойства в дочерних элементах без необходимости.
— Стили каждого блока держи рядом: классы должны быть сгруппированы в той же секции, к которой относятся элементы.
— Адаптивные стили выноси ТОЛЬКО после основных. Media queries группируй по брейкпоинтам и по секциям.
— Сохраняй порядок стилей таким же, как порядок блоков на странице.

.thank-you {
    min-height: 100vh;
    display: flex;
    align-items: center;
    justify-content: center;
    padding-block: 0;
    text-align: center;
    padding: 0 20px;
}
